# vue脚手架

## 说明
```
 vue webpack 自定义脚手架 支持多进程打包
 vue +webpack + vue-router + vux + less
```

## 安装依赖
```
yarn install & npm install
```

### 开发模式
```
yarn run dev & npm run dev
```

### 打包
```
yarn run build & npm run build
```

###  测试
```
yarn run test & npm run test
```

### 检测
```
yarn run lint & npm run lint
```
